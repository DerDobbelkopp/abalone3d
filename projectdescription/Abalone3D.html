Tests

Es werden mehrere Tests notwendig sein.


Test „Viele einzelne Kugeln zu demselben Ziel schicken“

Wenn viele einzelne Kugeln zur selben Zielposition sollen, werden nur wenige die Zielposition tatsächlich erreichen können. Die übrigen Kugeln werden versuchen die Kugeln von der Zielposition weg zu schieben. Hierbei können mehrere Fälle eintreten, z.B. können Kugeln (vor allem die, die das Ziel bereits erreicht haben) in einen Abgrund gedrückt werden.

Die Kugeln müssen also ein Cluster (oder Standardofrmation) bilden.

Effekt Cluster anzeigen, bevor Kugeln bewegt werden


Test „Travelling Marble Problem“

Kürzeste-Wege-Algorithmus soll vereinfacht werden, indem Standard-Kartenelementen vorab ein Wert zugewiesen wird. Auf dem Graphen der Karte wird dann der kostengünstigste Weg berechnet.
